Data Journalism:
Beers in the U.S.

Overview

For this project, I scraped data from Beer Advocate's 'Top 250 Beers' list using Ruby. I downloaded the data into Excel and sorted the data into categories based on location, beer type and brewery. I found that nearly 1/4 of the beers were in California, although the most popular brewery on the list hailed from Vermont.

Tools Used

I handed coded the page with HTML and CSS and created the charts using Illustrator.
